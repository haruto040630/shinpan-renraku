# 審判確認連絡アプリ 取扱説明書

東海学生サッカーリーグの審判員連絡を管理するWebアプリです。スマートフォン・PCから利用でき、複数端末間でデータがリアルタイム共有されます。

## 🔐 ログイン

1. URL を開く
2. パスワード `ymd69` を入力
3. 右の 👁 アイコンをタップで表示・非表示切替
4. 「ログイン」をタップ

---

## 🏠 ダッシュボード（セッション一覧）

ログイン後の画面。過去に作成したセッション（試合節）が一覧表示されます。

### セッションの操作ボタン

| ボタン | 機能 |
|---|---|
| **開く** | セッション詳細（試合カード）を表示 |
| **✏** | セッション情報を編集（年月日・備考・Excel差し替え） |
| **🗑** | セッションを削除 |

### 新規セッション作成

「**＋ 新規セッション作成**」をタップ：

1. **大会名**：「東海学生リーグ」または「東海学生トーナメント」を選択
2. **年・月**：プルダウンで選択
3. **第1日・第2日**：プルダウンで選択（第2日は任意）
4. **備考**：自由記入（任意）
5. **Excelファイル**：割当表をアップロード
6. 「セッションを作成して開く」をタップ

## ⚽ セッション詳細（試合カード）

### 都道府県タブ
「すべて / 愛知 / 岐阜 / 三重 / 静岡」で絞り込み表示。

### 試合カード構成

各試合に以下が表示されます：

- **対戦カード**（チームA vs チームB）
- **日付・KO時刻**
- **会場名・住所**（住所Excelとの自動マッチング）

#### 👤 会場責任者プルダウン
30名のリストから選択。選択すると **出力1の「会場責任者 未定／電話番号：未定」が自動で氏名・電話番号に置き換わります**。

#### 📤 送信元プルダウン
**山田・冨永・鈴木・籔中・小澤** から選択。**出力1の文末に1行空けて追記されます**。

### 審判ごとの操作（主審・AR1・AR2）

| ボタン | 機能 |
|---|---|
| **📋 出力1コピー** | この審判向けの個別連絡メッセージをクリップボードにコピー |
| **送信済み** | 送信完了をマーク（タップでON/OFF、緑色） |
| **確認済み** | 確認返信受領をマーク（タップでON/OFF、赤色） |

電話番号をタップで電話発信できます。

### 各試合の下部
**📞 出力3（電話番号リスト）コピー** ボタン

## 📝 出力1（個別連絡メッセージ）

審判員1人ずつに個別送信するための連絡文。LINEなどに貼り付けてそのまま送れます。

### 出力例

```
山田 様

お世話になっております。

今週末の東海学生リーグの割り当ての確認の連絡になります。

5/30（土）
名城大学第2グラウンド
住所：愛知県名古屋市天白区天白町大字八事裏山67-56
11:00ko
○○大学 対 △△大学

主審：山田 太郎 様
AR1：佐藤 二郎 様
AR2：田中 三郎 様

会場責任者 政岡将樹
電話番号：080-6945-0941

お忙しい中お手数おかけいたしますが、ご確認の返信を宜しくお願い致します。

冨永
```

- **冒頭の苗字**：その審判員の苗字が自動で入る
- **挨拶文の大会名**：選択した大会（リーグ／トーナメント）に応じて変わる
- **会場責任者**：プルダウンで選んだ責任者の氏名・電話番号
- **文末の送信元**：プルダウンで選んだ送信元名

会場責任者・送信元を**選択していない場合**は、それぞれ「未定」「（なし）」になります。

## 📝 出力2（確認状況一括報告）

画面上部の **「📝 出力2（確認済み報告）全試合コピー」** ボタンで、現在表示中の全試合分をまとめてコピー。

各審判の状態に応じて表記が自動で切り替わります：

| 状態 | 表記 |
|---|---|
| 送信済み ✓ ＋ 確認済み ✓ | **`/確認済み`** |
| 送信済み ✓ のみ | **`/メッセージ返信待ち`** |
| どちらも未反応 | **`/未対応`** |

### 出力例

```
名城大学第2グラウンド
5月30日
11:00ko
○○大学 対 △△大学
主審：山田 太郎 様 /確認済み
AR1：佐藤 二郎 様 /メッセージ返信待ち
AR2：田中 三郎 様 /未対応

━━━━━━━━━━━━
```

## 🔄 リアルタイム共有

すべての操作は **5秒以内** に他端末へ自動反映されます。

- **同期対象**：セッション作成・削除・編集、状態変更（送信済み・確認済み）、会場責任者選択、送信元選択
- **アプリバー右上の「🔄 同期中」**：同期中の表示。**消えていなくても他の操作は普通に行えます**（ノンブロッキング）
- **スマホスリープ中**：30秒間隔に自動切替（API節約）

## 📲 PWA（ホーム画面追加）

通常のWebアプリではなく **インストール可能なアプリ** として使えます。インストール後はアイコンから直接起動でき、アドレスバーなしのフルスクリーン表示になります。

### iPhone (Safari)

1. URL を **Safari** で開く（Chrome 不可）
2. 画面下の **共有ボタン**（↑のアイコン）をタップ
3. メニューをスクロールして **「ホーム画面に追加」** をタップ
4. 「追加」をタップ

### Android (Chrome)

1. URL を **Chrome** で開く
2. 右上のメニュー（⋮）から **「ホーム画面に追加」** または **「アプリをインストール」** をタップ
3. 「インストール」をタップ

## 🛠️ トラブルシューティング

### ログインできない
- パスワードを再確認（`ymd69`）
- 👁 アイコンで入力内容を表示確認
- ネット接続を確認
- PWAインストール済みの場合、一度アプリを終了して再起動

### 同期されない / データが見えない
- 同期中の表示が消えない場合は通信状況を確認
- 一度ログアウト → 再ログインで強制再同期

### Excelが読み込めない
- 拡張子が **`.xlsx`** であることを確認
- ファイルが破損していないか確認
- 別端末で同じファイルを試して切り分け

### 編集ボタン（✏）が反応しない
- ページをリロード（PWAは一度終了→再起動）
- スマホブラウザの場合、強制再読み込み

### アプリの内容が古い
- PWAインストール済みの場合：アプリ終了 → 再起動でキャッシュ更新
- ブラウザの場合：再読み込み（Chrome: ⋮ → 再読み込み）

## 📊 入力Excelの仕様

### 審判員割当 Excel
**1試合 = 2行1セット** の構造：
- 偶数行：試合情報＋審判員氏名
- 直下の奇数行：各審判員の電話番号

| 列 | 内容 |
|---|---|
| A | 大会名 |
| B | 試合名 |
| C | 開催日 |
| D | キックオフ時刻 |
| E | 会場名 |
| F | チームA |
| G | チームB |
| H | 主審 氏名 |
| I | 副審1 氏名 |
| J | 副審2 氏名 |
| K | アセッサー 氏名 |

会場名は内部の住所辞書と自動でマッチングされます（多少の表記揺れは吸収）。マッチしない場合は「住所未定」と表示されます。

## 🔒 セキュリティ

- **アプリパスワード認証**：全ユーザー共通
- **GitHubトークン**：AES-256-GCM 暗号化（パスワードで復号）
- **通信**：GitHub Pages の HTTPS による全通信暗号化
- **データ保存先**：GitHub プライベートでない公開リポジトリですが、パスワードなしには操作できません

---

## 📦 技術構成

- **フロントエンド**：単一HTMLファイル（SheetJS埋込）
- **ホスティング**：GitHub Pages（無料・永続）
- **データストア**：GitHub リポジトリ内の `data.json`
- **同期**：GitHub REST API
- **オフライン**：Service Worker（PWA）
